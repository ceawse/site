# AlpenStark Bank Frontend Clone

This project is a frontend clone of the AlpenStark Bank website, built with React, Vite, and Material UI.

## Tech Stack
- **React**: UI Library
- **Vite**: Build Tool
- **Material UI (MUI)**: Component Library
- **Formik & Yup**: Form handling and validation
- **React Router**: Client-side routing

## Getting Started

1.  Install dependencies:
    ```bash
    npm install
    ```

2.  Run the development server:
    ```bash
    npm run dev
    ```

3.  Open http://localhost:5173 in your browser.

## Project Structure
- `src/components`: Reusable UI components (Navbar, Footer).
- `src/pages`: Page components (Home, Register, Login, Dashboard).
- `src/theme`: MUI Theme configuration.
- `src/main.jsx`: Entry point.

## Implemented Features
- **Homepage**: Landing page with hero section.
- **Registration**: Multi-step account creation form with validation (Phone, Email, Password, Personal Details).
- **Navigation**: Responsive navbar and routing.
