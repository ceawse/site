# Руководство по проекту AlpenStark Bank

Этот файл содержит инструкции по установке, настройке и управлению платформой AlpenStark Bank.

## 1. Запуск проекта

### Инструкция по запуску:
1. Убедитесь, что у вас установлены Docker и Docker Compose.
2. В корневой директории проекта выполните команду:
   ```bash
   docker-compose up --build
   ```
3. После завершения сборки:
   - **Frontend (Сайт)** будет доступен по адресу: `http://localhost:8080`
   - **Backend (API)** будет работать на: `http://localhost:5000`

---

## 2. Настройка системы

### Настройка почты (SMTP)
Система использует `nodemailer` для отправки писем с подтверждением. Настройки находятся в файле [`server/server.js`](server/server.js).

Чтобы настроить свою почту, измените параметры в блоке `transporter` (строка 34) или установите соответствующие переменные окружения:
- `EMAIL_HOST`: SMTP-хост (например, `smtp.gmail.com`)
- `EMAIL_PORT`: Порт (обычно `587` или `465`)
- `EMAIL_USER`: Ваш email
- `EMAIL_PASS`: Ваш пароль (для Gmail это должен быть "Пароль приложения")

### База данных
Используется SQLite. Файл базы данных `database.sqlite` создается автоматически. Начальные настройки (адреса кошельков, контакты) задаются в файле [`server/db.js`](server/db.js).

---

## 3. Админ-панель и управление

### Как стать администратором
В системе настроена автоматическая выдача прав администратора для определенного email-адреса при регистрации.
- **Email админа по умолчанию**: `admin@alpenstark.com` ([`server/server.js:82`](server/server.js:82))

Чтобы получить доступ к админке:
1. Зарегистрируйте новый аккаунт, используя почту `admin@alpenstark.com`.
2. После входа у вас появится доступ к админ панеле (/admin в url).
